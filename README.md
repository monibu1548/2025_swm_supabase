# 🧑‍💻 2025 소프트웨어 마에스트로 16기 - 1인 서비스 개발 (Supabase)

## 🛠 기술 스택

- **Supabase**: 오픈소스 Firebase 대안, 인증/DB/파일 관리 지원
- **Next.js** (별도 저장소): 프론트엔드 프레임워크
- **PostgreSQL**: Supabase 기본 DB
- **TypeScript**, **REST API**, **Client SDK**




## 🧭 진행 방식

1. Supabase 프로젝트 생성 및 초기 설정
2. 인증, DB, 스토리지 연동 실습
3. Next.js 앱과 연동하여 CRUD 구현
4. 배포 및 실서비스 적용 전략
